Google Drive has one major blind spot — it never shows you how large your folders are. **GDrive Folder Size** fixes that.

| Feature | Description |
|---|---|
| 📊 **Real-Time Calculation** | See the exact total size of your entire Google Drive as it calculates — live, with no delays or manual refreshes. |

## 🚀 How to Use

1. **Install** the extension (see [Manual Install](#-manual-install) below).
2. **Open** [Google Drive](https://drive.google.com) in your Chrome browser.
3. **Click** the extension popup icon in your browser toolbar.
4. The extension will immediately begin calculating your **entire Google Drive size** in real time.
5. To check a **specific folder**, navigate into that folder, then open the popup again.
6. Done! 🎉 No login, no setup, no fuss.

## 🔧 Manual Install

Since this extension is currently in **Beta** and not yet on the Chrome Web Store, you need to install it manually:

### Step 1 — Download

[![Download ZIP](https://img.shields.io/badge/Download-.zip%20file-blue?style=flat-square&logo=googlechrome)](https://pixonexit.com/extensions/gdrive-folder-size/drive-folder-size-extension.zip)

Download the extension `.zip` file from the link above.

### Step 2 — Extract

Unzip the downloaded file to a folder on your computer. Remember where you save it.

### Step 3 — Open Chrome Extensions

1. Open **Google Chrome**
2. Go to `chrome://extensions/` in the address bar, **OR**
3. Click the three-dot menu → **More tools** → **Extensions**

### Step 4 — Enable Developer Mode

In the top-right corner of the Extensions page, toggle on **Developer mode**.

### Step 5 — Load the Extension

1. Click **"Load unpacked"**
2. Select the folder where you extracted the extension files
3. Click **Open**

### Step 6 — You're Ready! 🎉

The **GDrive Folder Size** icon will appear in your Chrome toolbar. Open Google Drive and click the icon to get started.
